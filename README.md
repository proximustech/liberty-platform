# Liberty Platform

**A flexible, low-complexity admin framework for Node.js with Frontend multi-framework support**

![TypeScript](https://img.shields.io/badge/TypeScript-Yes-blue)
![Frontend](https://img.shields.io/badge/Frontend-EJS%20%7C%20React%20%7C%20Vue-orange)

## Overview

Liberty Platform is a highly flexible admin enterprise framework that stands out by offering **zero frontend framework lock-in** and **low operational complexity**. Unlike other admin frameworks that force you into React-only solutions, Liberty Platform allows React, Vue, EJS and Vanilla JS, to coexist seamlessly within the same application.

## 🌟 Key Features

### ✨ No Framework Lock-in
- **Multi-framework support**: Use React, Vue, EJS, Vanilla JS. Any per plugin
- Mix and match frontend technologies based on your needs

### 🔐 Enterprise-Grade Security
- **Authentication**: Passport.js: local auth, OIDC, etc...
- **Permissions**: Casbin-based RBAC with attribute-level control
- Fine-grained access control ready to use

### 🧩 Flexible Plugin System
- Custom **IDynamicView** plugin architecture for Javascript Frontends
- Backend Middleware architecture for backend plugins
- Extend functionality without constraints

###  Performance & Simplicity
- Full SSR without client-side hydration overhead
- Faster time-to-interactive with a well designed architecture

##  Why Choose Liberty Platform?

### ✅ Perfect for:

- **Development of internal Enterprise applications**
- Solutions that want to implement functionalities through plugins
- Teams with any frontend expertise ( React/ Vue/ EJS / Vanilla JS)
- Applications needing maximum flexibility
- Custom admin panels with specific UI requirements
- Projects requiring gradual migration from legacy systems
- Access from Desktop and Mobile is needed
